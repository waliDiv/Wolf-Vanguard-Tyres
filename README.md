## About

Wolf Vanguard Tyres delivers **high-quality tyre solutions** for cars, SUVs, and trucks.  
The website is designed to:

- Build client trust with a premium design.  
- Showcase tyre products and services effectively.  
- Provide smooth navigation, responsive layout, and interactive elements.  
- Offer easy communication via phone, WhatsApp, and contact forms.

---

## Features

- ✅ **Responsive Navigation Bar** with hamburger menu for mobile.  
- ✅ **Hero Section** with gradient overlay and call-to-action button.  
- ✅ **About Section** highlighting company expertise and features.  
- ✅ **Services Section** with professional icons.  
- ✅ **Products Section** showcasing tyres with hover effects.  
- ✅ **Contact Section** with form and WhatsApp integration.  
- ✅ **Professional Footer** with quick links, services, and newsletter subscription.  
- ✅ Smooth scrolling animations and interactive buttons.  
- ✅ Modern dark theme with red accent for premium aesthetics.

## Technologies

- **HTML5** – Semantic structure  
- **CSS3** – Flexbox, Grid, gradients, transitions  
- **JavaScript** – Responsive menu toggle, smooth scroll, interactive alerts  
- **Google Fonts** – Inter font for modern typography  
- **Font Awesome** – Icons for services, phone, and buttons
